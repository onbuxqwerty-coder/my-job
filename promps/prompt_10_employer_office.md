# Промпт: Розробка кабінету роботодавця

## Цель проекту
Створити веб-додаток кабінету роботодавця для управління процесом рекрутингу, вакансіями та кандидатами.

---

## 1. Архітектура проекту

### Стек технологій:
- **Frontend**: React 18+ з Tailwind CSS
- **Backend**: Node.js + Express.js
- **База даних**: PostgreSQL
- **Аутентифікація**: JWT tokens
- **API**: RESTful API
- **Деплой**: Docker контейнери

## 4. Користувацькі сценарії

### Сценарій 1: Публікація вакансії
1. Роботодавець входить у кабінет
2. Переходить на сторінку "Мої вакансії"
3. Натискає "Створити нову вакансію"
4. Заповнює форму з інформацією про позицію
5. Натискає "Опублікувати"
6. Вакансія з'являється на дошці оголошень

### Сценарій 2: Перегляд та оцінка кандидатів
1. Роботодавець переходить до вакансії
2. Бачить список всіх кандидатів
3. Натискає на кандидата, щоб переглянути резюме
4. Додає нотатки та оцінку
5. Змінює статус на "Запрошений"
6. Відправляє запрошення на співбесіду

### Сценарій 3: Аналіз звітів
1. Роботодавець переходить на дашборд
2. Переглядає метрики та графіки
3. Бачить воронку конвертації
4. Експортує звіт у PDF
