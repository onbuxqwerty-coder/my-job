## ✅ Що тестується

### 🧪 Юніт-тести (10)
- Фільтрування кандидатів
- Пошук по ФІ та email
- Парсинг шаблонів повідомлень
- Роботу з датами

### 🔗 Інтеграційні (8)
- Таблиця кандидатів (завантаження, фільтри, сортування)
- Деталі кандидата (резюме, статус, нотатки)
- Система нотаток (додавання, редагування)
- Комунікація (повідомлення, шаблони)

### 🌐 API (11)
- GET /api/candidates (список, фільтри)
- POST /api/candidates (створення)
- PATCH /api/candidates/:id/status (зміна статусу)
- POST /api/candidates/:id/notes (нотатки)
- Обробка помилок (401, 404, 500)

### 🎬 E2E (18)
- Повний цикл рекрутингу (логін → вакансія → кандидат → собесіда)
- Планування собесід (календар, час, учасники)
- Пошук та фільтрування
- Експорт звітів
- Дашборд та аналітика

### ⚡ Performance (5)
- Таблиця 1000+ рядків завантажується за < 3s
- Пошук за < 500ms
- Сортування за < 1s
- Нема memory leaks

### ♿ Доступність (7)
- ARIA labels на всіх елементах
- Клавіатурна навігація (Tab, Enter, Escape)
- Контраст кольорів (WCAG AA)
- Screen reader сумісність

## 🆘 Помилки та Рішення

### ❌ "Cannot connect to API"
```
ECONNREFUSED: connect ECONNREFUSED 127.0.0.1:3000
```
**Рішення:** Запустіть сервер в новому терміналі
```bash
npm run dev
```

### ❌ "Browser not found"
```
Error: Executable doesn't exist at /path/to/chromium
```
**Рішення:** Установіть браузери
```bash
npx playwright install
```

### ❌ "Cannot find module"
```
Error: Cannot find module 'vitest'
```
**Рішення:** Установіть залежності
```bash
npm install --save-dev vitest
```
