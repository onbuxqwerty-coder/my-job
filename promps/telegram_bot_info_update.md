## 🔧 Крок за кроком: Як редагувати в BotFather

### **Крок 1: Відкрити BotFather**
1. Перейди до [@BotFather](https://t.me/botfather)
2. Натисни `/start`
3. Натисни `/mybots`
4. Вибери `@myjob_in_bot`

#### **2.4 Description Picture**
**Команда:** `/setdescriptionpic`

**Рекомендовані розміри:** 640x360 px (PNG або JPG)

**Концепція:**
- Громадське зображення, що показує платформу в дії
- Варіанти:
  1. **Скриншот головної сторінки** (список вакансій + фільтри)
  2. **Колаж із 4 блоків:**
     - Пошук роботи
     - Для роботодавців
     - Чати
     - Аналітика
  3. **Героїчна картинка:** "My Job — Твоя робота чекає!" з українською символікою

**Рекомендація:** Використай коїр: градієнт від синього (#0088cc) до фіолетового (#6B46C1), на якому лежить текст "My Job" та Telegram іконка.

---

#### **2.5 Bot Picture**
**Команда:** `/setuserpic`

**Рекомендовані розміри:** 512x512 px (PNG прозоре тло або JPG)

**Концепція:**
- Логотип My Job (найсуттєвіший, запам'ятовуючий)
- Умовно: літери "MJ" у квадраті або іконка вакансій
- **Палітра:** синій (#0088cc) + білий
- **Стиль:** сучасний, мінімалістичний, професійний

**Опція 1 (простий логотип):**
```
Білий квадрат (512x512)
Синій текст "MJ" (bold, 300pt, за центром)
Білі батончики (як лінії вакансій) під текстом
```

**Опція 2 (з Telegram інтеграцією):**
```
Комбінована іконка:
- Telegram логотип (50%)
- Портфель/вакансія іконка (50%)
- На синьому фоні з градієнтом
```

#### **2.6 Commands**
**Команда:** `/setcommands`

**Поточні команди:** 2 commands (перевір, які вони)

**Рекомендовані команди для My Job:**

```
/start — Розпочати авторизацію через My Job
/help — Отримати довідку та посилання
/vacancies — Переглянути топ вакансій
/profile — Управління профілем
/support — Контакти служби підтримки
/settings — Налаштування сповіщень
```

**Формат для BotFather:**
```
start — Розпочати авторизацію в My Job
help — Довідка та корисні посилання
vacancies — Топ вакансій
profile — Мій профіль
support — Служба підтримки

```

1. **Копіюєш один текст (About/Description/Privacy)**
