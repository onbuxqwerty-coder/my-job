Звіт: Функціональність бек-офісу роботодавця                                                                                                                                                                                      
                                                                                                                                                                                                                                    
  URL: /dashboard/employer | Middleware: auth, role:employer                                                                                                                                                                        
                                                                                                                                                                                                                                    
  ---                                                                                                                                                                                                                               
  1. Дашборд /dashboard/employer                                  

  - Перелік усіх вакансій компанії з метриками
  - Статистичні лічильники: всього вакансій / активних / відгуків
  - Активувати / деактивувати вакансію (toggleActive)
  - Видалити вакансію з підтвердженням
  - Посилання на редагування кожної вакансії
  - Відображення міста, типу зайнятості, дати публікації

  ---
  2. Профіль компанії /dashboard/employer/profile

  - Редагування: назва, опис, вебсайт, місто (CitySearch)
  - Slug генерується автоматично з назви
  - Збереження через VacancyService / безпосередньо в модель

  ---
  3. Вакансії /dashboard/employer/vacancies/create та /edit/{id}

  Єдина форма для створення та редагування:
  - Заголовок, опис (rich text)
  - Тип зайнятості (EmploymentType enum)
  - Категорія (з БД)
  - Місто (CitySearch компонент)
  - Зарплата: від / до / валюта
  - Мови, придатність (Language, Suitability enums)

  ---
  4. Відгуки на вакансію /dashboard/employer/applicants/{vacancyId}

  - Перелік кандидатів по конкретній вакансії
  - Фільтрація за статусом (pending, screening, interview, hired, rejected)
  - Лічильники по кожному статусу
  - Зміна статусу прямо зі списку (updateStatus)
  - Нотатки до кандидата (saveNote)
  - Перехід до детальної картки

  ---
  5. Всі кандидати /dashboard/employer/candidates

  - Зведений список по всіх вакансіях
  - Фільтри: пошук, статус, вакансія, період, рейтинг
  - Сортування: по імені, рейтингу, даті
  - Пагінація (25 на сторінку)
  - Виставлення рейтингу (updateRating) прямо зі списку
  - Очищення фільтрів одним кліком

  ---
  6. Деталь кандидата /dashboard/employer/candidates/{applicationId}

  - Повна інформація про кандидата та його заявку
  - Зміна статусу заявки
  - Виставлення рейтингу (зірки)
  - Нотатки: додавання, редагування, видалення
  - Планування співбесіди: дата, час, тривалість, тип (online/offline), посилання, адреса, нотатки
  - Скасування запланованої співбесіди
  - Надсилання повідомлення кандидату: вибір шаблону, тема, тіло листа, копія відправнику

  ---
  7. Шаблони повідомлень /dashboard/employer/templates

  - Створення / редагування / видалення шаблонів
  - Типи: MessageType enum (запрошення, відмова тощо)
  - Активація / деактивація шаблону
  - Перелік доступних змінних ({{name}}, {{vacancy}} тощо)

  ---
  8. Аналітика /dashboard/employer/analytics

  - Вибір періоду: 7д / 30д / 90д / рік
  - Статистика відгуків по статусах
  - Активність по кандидатах (нотатки, повідомлення, співбесіди)
